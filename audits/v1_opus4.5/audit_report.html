High Impact Factors (Could shift results >20%)

Space Radiation Effects on GPUs VERY HIGH IMPACT

LEO radiation flux is ~100x terrestrial. The model uses Meta's 9%/year ground-based GPU failure rate.

  • Space radiation could increase failure rate 2-5x
  • Radiation hardening adds mass (10-30% of compute mass)
  • Single-event upsets require error correction overhead

Estimate: Could add 20-50% to effective orbital costs

Manufacturing Learning Curve HIGH IMPACT (Positive)

Model uses current satellite costs. At 40,000+ unit production:

  • SpaceX demonstrates ~85% learning curve on Starlink
  • Could reduce $/kg by 20-50%
  • Already partially reflected in V2 Mini vs V1 costs

Estimate: Could reduce orbital costs 15-30% at scale

Discount Rate / Cost of Capital HIGH IMPACT

Model treats all costs as undiscounted (NPV = sum of costs).

  • Orbital: All costs front-loaded (launch + hardware upfront)
  • Terrestrial: Fuel costs spread over 5 years
  • At 10% WACC, NPV of terrestrial fuel is ~15% lower

Estimate: Discounting slightly favors orbital (5-10%)

Technology Obsolescence HIGH IMPACT

GPU architecture changes every 2-3 years. 5-10 year analysis assumes stable compute.

  • Terrestrial: Easy hardware refresh cycle
  • Orbital: Must launch replacement satellites
  • Stranded asset risk much higher for orbital

üìù Recommendations

Model Improvements

  1. Add radiation effects model - Include radiation-adjusted GPU failure rates and shielding mass
  2. Add battery mass for non-terminator orbits - Current model assumes 98% sun fraction
  3. Add discount rate / NPV calculation - Front-loaded vs distributed costs matter
  4. Add station-keeping propellant budget - ~5-10% of dry mass for 5-year LEO
  5. Add communications throughput constraint - May limit viable workloads

Analysis Extensions

  1. Monte Carlo sensitivity - Run 10,000 scenarios with parameter distributions
  2. Learning curve model - Project costs at 10x, 100x production scale
  3. Carbon price scenarios - Test $50-200/ton CO2 impact on terrestrial
  4. Alternative architectures - Test dedicated radiator mass vs bifacial
